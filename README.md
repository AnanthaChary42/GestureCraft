# Hand Gesture Hologram
## Setup
1. **Python**: Ensure Python 3.8 - 3.11 is installed.
2. **Installation**: Run the setup script to install dependencies and download the model.
   ```bash
   python setup_project.py
   ```
3. **Run**:
   ```bash
   python backend/main.py
   ```
